ID-AL JavaScript SDK version 1.0.0
-----------------------------------

The ID-AL JavaScript SDK is a JavaScript development kit giving access to the
ID-AL VP320, VP330 or EVP380 video player features.

The zip archive contains production and development versions (minified or not).
The development versions contain stubbed implementations that simulate the
player APIs when running on a PC.

Production versions:
 - idal-sdk.min.js
 - idal-sdk.js

Development versions:
 - idal-sdk.dev.min.js
 - idal-sdk.dev.js
